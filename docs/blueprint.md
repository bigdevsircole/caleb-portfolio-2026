# **App Name**: PixelForge Portfolio

## Core Features:

- Dynamic Project Showcase: Display a grid of portfolio projects with details (Title, Description, Image, Link) fetched and managed via Firebase Firestore.
- Digital Products Sales Page: A dedicated section to list and promote courses and digital tools, each featuring a 'Buy Now' button placeholder for future integration with a checkout flow. Product data is managed in Firebase Firestore.
- Secure Resource Library: A protected area where customers can access downloadable files or exclusive links. Resource metadata and access permissions will be managed through Firebase Firestore.
- Interactive Hero Section: Display your name prominently with a CSS Neon Glow effect, complemented by a typing animation cycling through 'Frontend Developer', 'Designer', and 'IT Support'.
- Reveal-on-Scroll Animations: Integrate Framer Motion to create engaging 'reveal on scroll' effects for all main sections of the website, enhancing user experience.
- Subtle Click Feedback: Implement Framer Motion to add subtle scale-down or ripple effects when interactive elements like buttons or project cards are clicked, providing tactile feedback.
- Firestore Data Management: Utilize Firebase Firestore as the backend database to efficiently store and retrieve all project details, product information for the sales page, and customer-exclusive resource data.

## Style Guidelines:

- Color scheme: Predominantly dark mode by default to enhance neon effects and content contrast. Primary: Vibrant electric purple (#934CE6) to provide a modern, high-tech glow for headlines and interactive elements. Background: Deep charcoal purple (#1A161F) for the main page content, visibly desaturated from the primary hue. Accent: Bright, energetic blue (#5678F7) for subtle highlights, interactive states, and call-to-action elements, analogous to the primary color.
- Headlines and body text: 'Space Grotesk' (sans-serif) for a modern, techy, and clean aesthetic that complements the digital and neon-themed visual identity, suitable for both bold statements and readable content.
- Background: Animated motion of floating syntax patterns to reinforce the tech-centric theme. Iconography: Clean, minimalist, and outline-based icons to maintain a sophisticated and modern digital aesthetic, ensuring they pop against the dark background.
- Sections are designed for optimal content hierarchy, utilizing a flexible grid layout for projects and a clear, modular approach for product listings, ensuring responsiveness across devices.
- Global animations leverage Framer Motion for smooth transitions, reveal effects, and subtle micro-interactions, contributing to a dynamic and polished user experience without feeling distracting. This includes the specified typing animation and scroll-based content reveals.
